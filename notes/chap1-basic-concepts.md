### Define the following terms:

| term                 | define                                                                                                                                                            |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| logic                | the organized body of knowledge, or science, that evaluates arguments.                                                                                            |
| argument             | a group of statements                                                                                                                                             |
| statement            | a sentence that is either true or false                                                                                                                           |
| truth value          | Truth and falsity                                                                                                                                                 |
| premise              | the state ments that set forth the reasons or evidence                                                                                                            |
| conclusion           | the evidence is claimed to support or imply.                                                                                                                      |
| conclusion indicator | “for this reason”                                                                                                                                                 |
| premise indicator    | “for the reason that”                                                                                                                                             |
| inference            | the reasoning process expressed by an argument                                                                                                                    |
| proposition          | the meaning or information content of a statement                                                                                                                 |
| syllogistic logic    | a kind of logic in which the fundamen tal elements are terms, and arguments are evaluated as good or bad depending on how the terms are arranged in the argument. |
| modal logic          | a kind of logic that involves such concepts as possibility, necessity, belief, and doubt                                                                          |
